🚀 Hacknovation – Hackathon Team Collaboration App

📱 Overview

Hacknovation is a modern Android application designed to simplify hackathon team formation and collaboration. It enables users to discover, join, and manage teams efficiently through a clean and intuitive interface.

The app focuses on real-time interaction, smooth user experience, and scalable architecture, making it ideal for students and developers participating in hackathons.

---

✨ Features

- 👥 Create and Join Teams
- 🔍 Search & Filter Teams
- 📡 Real-time Data Sync (Firebase)
- 📱 Clean & Responsive UI/UX
- 🔐 User Authentication
- ⚡ Fast Performance & Optimized Code

---

🛠️ Tech Stack

- Language: Java / Kotlin
- IDE: Android Studio
- Backend: Firebase Realtime Database
- Authentication: Firebase Auth
- UI: XML + Material Design
- Version Control: Git & GitHub
